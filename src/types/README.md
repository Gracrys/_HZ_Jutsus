Purpose
-------
This folder holds shared TypeScript type definitions for the project.

Usage
-----
- Import types in code: `import type { Jutsu } from '$lib/types';` (adjust path or add a path mapping if desired).
- You can also add global declarations here (e.g. `global.d.ts`) if you need ambient types.

Tip
---
If you want to import types using `'$lib/types'` or similar aliases, consider adding or adjusting `paths` in `tsconfig.json`.
